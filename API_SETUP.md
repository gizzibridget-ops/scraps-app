# Recipe App - API Setup Guide

## Setting up Claude AI Recipe Generation

This app uses the Anthropic Claude AI API to generate personalized recipe suggestions based on your ingredients.

### Getting Your API Key

1. Visit [Anthropic Console](https://console.anthropic.com/)
2. Sign up or log in to your account
3. Navigate to API Keys section
4. Create a new API key
5. Copy your API key

### How It Works

When you add ingredients and click "Find Recipes", the app:
1. Collects all ingredient tags you entered
2. Sends them to Claude AI (claude-sonnet-4-20250514 model)
3. Claude generates 6 unique, creative recipes tailored to your ingredients
4. Each recipe includes:
   - Creative name and description
   - Food emoji
   - Cook time and servings
   - Difficulty level
   - Complete ingredient list (marked as "have" or "need to buy")
   - Step-by-step cooking instructions
   - Relevant tags for filtering

### Important Notes

- **API Key Required**: The app will show an error if you try to generate recipes without an API key
- **Different Results Every Time**: Claude generates fresh recipes each time based on your ingredients
- **No Hardcoded Recipes**: All recipes are dynamically generated by AI

### Privacy & Security

- Your API key is stored locally and never shared
- Ingredient data is only sent to Anthropic's API when generating recipes
- No personal data is collected or stored

### Note

This app is for demonstration purposes. Please be mindful of API usage costs and rate limits on your Anthropic account.
